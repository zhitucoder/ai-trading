# 股票画像与标签系统

基于 Mark Minervini 的投资理念构建股票标签体系，层层递进，用作 AI 生成代码的需求文档。

---

## 数据结构

- 每支股票在每个交易日生成一份完整画像
- 画像由一组标签（tag）组成
- 标签分两级：**指标类**（ind.*）→  **业务类**（biz.*），另加 **阶段类**（stage.*）
- 指标类是原始计算结果，业务类由多个指标类组合形成有业务意义的判断

```
基础数据 → 指标类标签 (ind.*) → 业务类标签 (biz.*)
                                        → 阶段标签 (stage.*)
```

---

## 一、指标类标签（ind.*）

原子条件，直接从数据计算产生 true/false 或数值。

### 1.1 均线类

| tag_id | 条件 |
|--------|------|
| `ind.ma5_above_ma10` | MA5 > MA10 |
| `ind.ma10_above_ma20` | MA10 > MA20 |
| `ind.ma20_above_ma60` | MA20 > MA60 |
| `ind.price_above_ma20` | 收盘价 > MA20 |
| `ind.price_above_ma50` | 收盘价 > MA50 |
| `ind.price_above_ma150` | 收盘价 > MA150 |
| `ind.price_above_ma200` | 收盘价 > MA200 |

### 1.2 RSI 类

| tag_id | 条件 |
|--------|------|
| `ind.rsi_overbought` | RSI14 > 70 |
| `ind.rsi_oversold` | RSI14 < 30 |

### 1.3 成交量类

| tag_id | 条件 |
|--------|------|
| `ind.volume_surge` | 当日成交量 > 20日均量 × 1.5 |
| `ind.volume_shrink` | 当日成交量 < 20日均量 × 0.5 |

### 1.4 原始数值类（供业务标签引用）

| tag_id | 数据来源 | 说明 |
|--------|---------|------|
| `ind.revenue_growth` | `fin_ratios.revenue_growth_rate` | 营收增长率（数值） |
| `ind.net_profit_growth` | `fin_ratios.net_profit_growth_rate` | 净利润增长率（数值） |
| `ind.debt_ratio` | `fin_ratios.debt_asset_ratio` | 资产负债率（数值） |
| `ind.contract_liab_growth` | `fin_ratios.contract_liability_growth` | 合同负债增长率（数值） |
| `ind.revenue_cagr_3y` | `fin_ratios.revenue_cagr_3y` | 营收 3 年复合增长率（数值） |
| `ind.profit_cagr_3y` | `fin_ratios.net_profit_cagr_3y` | 净利润 3 年复合增长率（数值） |

## 二、业务类标签（biz.*）

组合多个指标类标签，形成有业务意义的判断结果。

### 2.1 成长性标签

| tag_id | 组合条件 | 标签名 |
|--------|---------|--------|
| `biz.boom_growth` | `ind.revenue_growth` > 50% AND `ind.net_profit_growth` > 50% | 爆发增长 |
| `biz.high_growth` | `ind.revenue_growth` > 20% AND `ind.net_profit_growth` > 20% | 高成长 |
| `biz.steady_growth` | `ind.revenue_growth` > 0% AND `ind.net_profit_growth` > 0% | 稳健增长 |

### 2.2 下滑/衰退类标签

| tag_id | 组合条件 | 标签名 |
|--------|---------|--------|
| `biz.revenue_decline` | `ind.revenue_growth` < 0% | 营收下滑 |
| `biz.profit_decline` | `ind.net_profit_growth` < 0% | 利润下滑 |
| `biz.double_decline` | `ind.revenue_growth` < 0% AND `ind.net_profit_growth` < 0% | 营收利润双降 |
| `biz.profit_collapse` | `ind.net_profit_growth` < -50% | 业绩暴雷 |
| `biz.profit_to_loss` | 上期净利润 > 0，本期 < 0 | 由盈转亏 |
| `biz.growth_slowdown` | 上期增长率 > 30%，本期 < 10% | 增速大幅放缓 |

### 2.3 财务健康标签

| tag_id | 组合条件 | 标签名 |
|--------|---------|--------|
| `biz.financial_healthy` | `ind.debt_ratio` < 40% AND `ind.net_profit_growth` > 0% | 财务健康 |
| `biz.financial_moderate` | `ind.debt_ratio` BETWEEN 40% AND 60% | 负债适中 |
| `biz.financial_risky` | `ind.debt_ratio` > 60% | 高负债风险 |
| `biz.financial_critical` | `ind.debt_ratio` > 80% | 高杠杆 |
| `biz.financial_insolvent` | `ind.debt_ratio` > 100% | 资不抵债 |

### 2.4 趋势动量标签

| tag_id | 组合条件 | 标签名 |
|--------|---------|--------|
| `biz.strong_momentum` | `ind.price_above_ma200` AND `ind.ma20_above_ma60` | 趋势强劲 |
| `biz.weak_momentum` | NOT `ind.price_above_ma200` | 趋势疲弱 |
| `biz.bullish` | `ind.ma5_above_ma10` AND `ind.ma10_above_ma20` AND `ind.ma20_above_ma60` | 多头排列 |
| `biz.bearish` | NOT `ind.ma5_above_ma10` AND NOT `ind.price_above_ma20` | 空头排列 |
| `biz.volume_price_up` | `ind.volume_surge` AND `ind.price_above_ma20` | 量价齐升 |
| `biz.volume_price_down` | `ind.volume_surge` AND NOT `ind.price_above_ma20` | 放量下跌 |
| `biz.break_support` | 前日在 MA60 上方，今日跌破 MA60 | 破位下跌 |

### 2.4 连续增长标签

需跨季度对比，从 `fin_income` / `fin_ratios` 遍历历史报告期。

| tag_id | 条件 | 标签名 |
|--------|------|--------|
| `biz.consecutive_profit_3q` | 净利润连续 3 季度同比增长 > 0% | 连增 3 季 |
| `biz.consecutive_profit_5q` | 净利润连续 5 季度同比增长 > 0% | 连增 5 季 |
| `biz.consecutive_revenue_3q` | 营收连续 3 季度同比增长 > 0% | 营收连增 3 季 |

## 三、阶段类标签（stage.*）

基于 Stan Weinstein 四阶段理论 + Mark Minervini 趋势模板。

| 阶段 | 名称 | 核心特征 |
|------|------|---------|
| S1 | 打底蓄势期 | 股价围绕 200 日均线横盘，成交量低迷 |
| S1S2 | 过渡期 | 从底部爬升，站上均线，高低点逐步抬升 |
| S2 | 突围加速期 | 股价在 200 日均线上方，趋势向上，成交量放大 |
| S3 | 见顶派发期 | 波动加大，200 日均线走平，主力出货 |
| S4 | 衰败下跌期 | 股价在 200 日均线下方，持续新低 |

### 阶段判定逻辑

```
S2（优先判断，满足趋势模板即 S2）→ S1S2 → S4 → S3 → S1（默认）
```

### S2 判定 — 趋势模板（全部满足）：

1. 当前股价在 150 日和 200 日均线上方
2. 150 日均线在 200 日均线上方
3. 200 日均线至少上涨了 1 个月（交易日 ≥ 22）
4. 50 日均线高于 150 日及 200 日均线
5. 当前价格高于 50 日均线
6. 当前股价比最近一年最低价至少高 30%
7. 当前价格在最近一年最高价的 25% 以内

### S1S2 判定 — 过渡期（S1 → S2 过渡信号，全部满足）：

- 股价站上 150 日、200 日均线上方
- 150 日均线高于 200 日均线
- 200 日均线开始向上
- 最高价不断抬高，最低价稳步抬高
- 快速上涨周中成交量放大
- 上涨周数量多于下跌周

### S4 判定（任一满足）：

- 股价多数时间（60%+ 交易日）在 200 日均线下方
- 200 日均线明确下行（MA200 比 20 日前低）
- 股价靠近一年内最低点（距最低 < 10%）

股价多数时间运行在200 日均线下方；
200 日均线进入明确下行通道；
股价靠近一年内价格低点区间；
股价持续创出阶段新低，下跌呈台阶式走低；
短期均线运行在长期均线下方；
成交量放大的交易日 / 交易周里，股价下跌幅度更大；
放量周期中，下跌交易日、交易周数量多于上涨时段；
企业盈利增长动力衰退，利空消息逐步释放，机构下调估值评价

### S3 判定：

股价波动性明显上升，涨跌幅度持续变大；整体走势看似仍偏强，但震荡加剧；
常会出现放量大跌行情，多为二阶段启动后最大单周 / 单日跌幅，下跌伴随大额成交量；
股价常在 200 日均线上下反复震荡游走；
200 日均线上涨动能衰竭，逐步走平，随后拐头向下；
主力筹码由前期获利买方，逐步转手给跟风散户买方；个股热度提升、曝光度变高；
公司净利润增速放缓，业绩利好逐步兑现完毕。


### S1 判定：

在第一阶段，股票价格因缺少让其上涨或下跌的动力而不起眼地波动着。
股价会围绕 200 日均线上下波动。在波动中，它没有实质性的向上或向下的趋势。这种死水一般的时期可能会持续几个月甚至几年。
这个阶段通常发生在股价从第四阶段下跌几个月之后。
交易量通常很小，尤其小于之前的第四阶段。

### 阶段标签

| tag_id | 标签名 |
|--------|--------|
| `stage.s1` | 打底蓄势期 |
| `stage.s1s2` | 过渡期 |
| `stage.s2` | 突围加速期 |
| `stage.s3` | 见顶派发期 |
| `stage.s4` | 衰败下跌期 |

## 五、设计原则

1. **指标可扩展**：新增指标标签只需在 `definitions.py` 定义 `ind.*`，不影响已有标签
2. **业务可组合**：业务标签通过组合现有指标标签形成，无需重复计算
3. **画像可更新**：每日自动重算，保持最新状态
4. **分层递进**：基础数据 → 指标类标签（ind.*）→ 业务类标签（biz.*）+ 阶段标签（stage.*）
