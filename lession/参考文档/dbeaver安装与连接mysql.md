# DBeaver 安装与连接 MySQL 指南

> DBeaver 是一个数据库可视化工具，能让你像 Excel 一样看数据库里的数据
> **版权：** 开源免费软件（Apache 2.0 / GPL 协议），个人和商业使用均无版权问题

## 1. 下载安装

**下载地址：** https://dbeaver.io/download/

根据你的系统选择对应版本：

| 系统 | 版本 |
|------|------|
| Windows | 下载 Windows (installer) `.exe` |
| macOS | 下载 macOS (dmg) |
| Linux | 下载 Linux (deb/rpm) 或去应用商店搜索 DBeaver |

一路下一步安装即可。

## 2. 连接 MySQL

打开 DBeaver，点击菜单栏的 **"数据库" → "新建连接"**，或者点击左上角的 **"+" 插头图标**。

在弹出的窗口中选择 **MySQL**，点击下一步。

填写连接信息：

| 字段 | 填写内容 | 说明 |
|------|---------|------|
| **主机** | `localhost` 或 `127.0.0.1` | 数据库所在电脑的地址 |
| **端口** | `3306` | MySQL 端口（本课程用 3306） |
| **数据库** | `ai_trading` | 数据库名称 |
| **用户名** | `root` | 数据库账号 |
| **密码** | `aitrading123` | 数据库密码 |

点击 **"测试连接"**，如果提示 "已连接" 即成功。

如果提示需要下载驱动，点 **"下载"** 让它自动安装。

## 3. 常用操作

| 操作 | 方法 |
|------|------|
| **查看表数据** | 双击左侧表名（如 `stocks`、`daily_prices`） |
| **执行 SQL** | 右键数据库 → "SQL 编辑器" → 写 SQL → 按 Ctrl+Enter 执行 |
| **刷新数据** | 右键表 → "刷新" 或按 F5 |
| **查看表结构** | 双击表名展开，能看到字段名和类型 |

## 4. 查看数据库中的数据

连接成功后，左侧树形结构展开：
```
ai_trading
├── 表
│   ├── stocks          ← 股票信息表（代码、名称、交易所）
│   ├── daily_prices    ← 日K线数据表（开盘、收盘、最高、最低、成交量）
│   ├── financial_reports  ← 财务报表
│   ├── ...
```

双击 `daily_prices` 就能看到千万级的日K线数据，像 Excel 表格一样展示。
