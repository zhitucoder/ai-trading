# 中信证券交易软件安装指南

## 1. 下载

访问下载页面：
https://www.cs.ecitic.com/newsite/rjxz/dnkhd/

选择 **"中信证券至信新版"**（第一个下载链接）。

## 2. 安装

> **重要：安装目录不要包含空格**，否则后面程序解析路径会报错。

双击安装包，选择安装路径为（本课程的路径）：
```
D:\programs\stock
```

路径中不能有空格。其他选项保持默认，一路下一步完成安装。

## 3. 下载数据（不需要登录）

打开软件后，**不需要登录账号**，直接操作：

**下载日K线数据：**
右上角 **"设置" → "盘后数据下载"**，勾选需要的市场和日期范围，下载即可。

**下载财务报表数据：**
右上角 **"设置" → "专业财务数据"**，勾选需要的报告期，下载即可。

下载后的数据文件路径（本课程）：
```
D:\programs\stock\vipdoc\
```

## 4. 关于「通达信数据」

中信证券交易软件内置的就是通达信内核，下载的数据格式、文件结构、存储路径与通达信完全一致。

**本课程后续所有提到「通达信数据」的地方，指的就是中信证券安装目录下的数据。** 不需要额外安装通达信软件，使用中信证券即可。注意下载数据的路径是 `D:\programs\stock\vipdoc\`。
