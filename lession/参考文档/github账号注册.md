# GitHub 账号注册指南

## GitHub 是什么

GitHub 是全球最大的代码托管平台，简单说就是**代码的"百度网盘"+ "朋友圈"**——开发者把代码存到上面，既能备份和版本管理，也能跟其他人协作。

## 多少人用它

截至 2025 年，GitHub 拥有 **超过 1 亿注册用户**，托管了 **超过 4.2 亿个仓库**（项目），几乎所有的开源项目都在上面——Linux、Python、React、Vue、VS Code……全球绝大多数开发者的日常工作都离不开它。

## 为什么用 GitHub

- **备份代码**：代码存到云端，换电脑也不丢
- **版本管理**：每个改动都有记录，改错了随时回退
- **AI 编程的基础设施**：Claude Code / OpenCode 都在 GitHub 上读取和提交代码，配合 AI 编程工具使用 GitHub 是标准流程
- **学习资源**：上面有海量开源项目可以看别人的代码怎么写的
- **社交名片**：GitHub 主页就是程序员的"简历"，面试时拿得出手

简单说：**本课程后续所有代码都会通过 AI 工具提交到 GitHub，所以需要你先有一个 GitHub 账号。**

---

## 1. 打开注册页面

浏览器访问：https://github.com/signup

## 2. 填写信息

| 字段 | 说明 |
|------|------|
| **Email** | 填写你的常用邮箱 |
| **Password** | 设置密码（至少 8 位，含数字和字母） |
| **Username** | 输入你想要的用户名（别人能看到，建议用英文名或拼音） |
| **Verify** | 按提示完成拼图验证 |

## 3. 邮箱验证

提交后，GitHub 会发送一封验证邮件到你填写的邮箱。登录邮箱，点击 **"Verify email address"** 按钮完成验证。

## 4. 完成引导

验证后可能有一些简单的引导页面，直接跳过或按喜好选择即可。

## 5. 验证注册成功

打开 https://github.com，应该显示已登录状态，右上角能看到你的头像。

## 注意事项

- 建议用 Gmail / Outlook 等国际邮箱，国内部分邮箱可能收不到验证邮件（163邮箱可以用）
- 如果收不到邮件，检查垃圾邮件箱
- 注册不需要绑定手机号（除非开启双重验证）
- 用户名一旦注册不能修改（但有办法改，较为麻烦），建议慎重
