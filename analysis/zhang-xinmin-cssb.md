# 中船防务2025年报：张新民视角下的造船龙头财务分析

> *以张新民"战略视角财务报表分析"框架分析中船防务（600685）。中船防务是实体制造业企业，军工+造船双主业，属于我框架的适用范围内。以下分析基于2025年年报公开数据。*

---

## 一、先看战略：中船防务在做什么样的生意？

中船防务这个企业，首先得看清楚它的战略定位。它不是一个纯粹的造船厂——它是一头是军工、一头是民船的双轮驱动企业。控股子公司黄埔文冲，1851年建厂，175年历史，横跨三个世纪，是国内军用舰船和公务船的重要建造基地，同时也是全球领先的支线集装箱船制造商。

用我的"战略视角资产负债表重构"来看：中船防务是典型的**经营主导型**企业——它的核心资产是造船产能、生产线、技术工人，它的营业收入90%来自造船产品交付。它不靠投资赚钱，不靠金融操作，就是实实在在地造船卖船。

但是有一个特殊之处——中船防务本身是**控股型公司**。"专注于资产经营、投资管理"，真正的生产经营实体在子公司黄埔文冲。所以分析中船防务，必须把合并报表和母公司报表分开看。母公司更像一个战略管控平台，子公司在干实活。

## 二、看利润质量：净利润增长167%，但核心利润才是关键

2025年中船防务归母净利润10.08亿，同比增长167%。数字很漂亮，对不对？但按我的"核心利润中心论"，不能光看净利润，要把利润拆开来看：

按我的方法，核心利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 利息费用。我帮大家做个粗略拆分：

| 利润构成 | 金额（亿元） | 说明 |
|---|---|---|
| 营业收入 | 205.47 | 造船为主 |
| 减：营业成本 | 184.79 | 含材料、人工等 |
| **核心利润（估算）** | **约11.4** | 主业真实盈利 |
| 加：投资收益 | 8.35 | 主要来自联营企业广船国际 |
| 加：其他收益等 | 0.44 | 公允价值变动+其他 |
| 减：剩余费用等 | 约-9.0 | 管理/销售/财务费等 |
| **净利润** | **11.38** | 合并净利润 |
| 归母净利润 | 10.08 | |

大家注意一个关键数字：**投资收益8.35亿**，其中广船国际贡献了7.93亿。净利润11.38亿里，投资收益占了8.35亿，也就是73%。什么意思呢？中船防务自己造船赚的钱，和从联营企业分来的钱，后者贡献更大。

这就要问一个问题了：**中船防务的核心利润到底是多少？**

我们用更严格的口径来算。2025年造船产品收入184.72亿，毛利率10.27%，造船业务本身的毛利大约是18.97亿。减去三费（6.69亿）、研发费用（9.03亿），再考虑其他税金和利息，造船主业的净利润率其实很薄——大概在3-4%的水平。

所以我告诉大家，中船防务2025年的利润结构是这样的：
- **价值创造（核心利润）**：造船主业，大概贡献了净利润的60-70%
- **价值整合（投资收益）**：广船国际等联营企业贡献了近30%
- **其他浮动收益**：公允价值变动等，占比很小

这个结构在改善。2024年投资收益只有2.5亿左右，2025年增长到8.35亿，联营企业确实做得好。但我更关注的是造船主业——主业毛利率从9.32%提升到10.27%，集装箱船毛利率达到26.73%，这是真正健康的信号。

## 三、看资产负债表：经营资产为主，负债结构值得关注

截至2025年末，总资产560.22亿，负债343.31亿，资产负债率61.28%。

按我的框架，我先把右边拆开——看它的**造血能力和输血结构**：

**负债端结构分析（合并报表口径）：**

| 项目 | 金额（亿元） | 占负债比 |
|---|---|---|
| 合同负债（预收客户款） | 约65 | 18.9% |
| 应付票据及应付账款 | 约120 | 35.0% |
| 短期借款 | 44.1 | 12.8% |
| 长期借款 | 约22 | 6.4% |
| 其他流动负债+非流动负债 | 约92 | 26.9% |
| **负债合计** | **343.31** | **100%** |

大家看到没有？合同负债（也就是客户提前给的钱，造船行业的惯例）加上应付票据及应付账款，这两项合计占了负债的**54%**。这叫什么？这就是我讲的**"造血能力"**——客户愿意提前打钱，供应商愿意赊账，说明中船防务在产业链里有话语权。

但我也要说句实在话：短期借款44.1亿，同比增加了327.51%，这个增速让人有点紧张。64亿的总债务里，短期债务占66.53%。用短期借款支持造船这种长周期业务——从订单到交付一般2-3年——存在期限错配的风险。

看左边——**资产端结构**：

| 项目 | 金额（亿元） | 占比 |
|---|---|---|
| 货币资金 | 约53 | 9.5% |
| 存货 | 85.28 | 15.2% |
| 应收账款+应收票据 | 约40 | 7.1% |
| 合同资产 | 约30 | 5.4% |
| 固定资产 | 45.53 | 8.1% |
| 在建工程 |约15 | 2.7% |
| 长期股权投资（广船国际等） |约80 |14.3% |
| 其他 | 约211 |37.7% |
| **总资产** | **560.22** | **100%** |

存货85.28亿，同比增长44.41%。存货占净资产（216.91亿）的46.77%——差不多一半的净资产变成了船台上的钢板和半成品。这让我想起了我讲课时常说的那句话："固定资产-营收-核心利润三者必须匹配。"

中船防务的情况是：存货大增44%，但营收只增长5.9%。这就是一个**"三个脱节"的信号**——当然，造船行业有特殊性，在手订单606亿，未来两三年都要出货，存货增长反映的是"为了交付在备料"，不一定是有问题的信号。但需要警惕的是：存货跌价准备只有1.94亿，计提比例2.22%，这个比例对于船舶行业来说够不够？如果钢价下跌或者船型改型，这部分存货有没有减值风险？

## 四、看三个匹配：关键信号分析

### 匹配一：固定资产与营业收入

固定资产45.53亿，与2024年基本持平。营业收入205.47亿，增长5.9%。固定资产没有大幅扩张，营收在增长——说明产能利用率在提升，这是好信号。主建船型批量化建造，生产效率提升，这条路径是对的。

### 匹配二：营业收入与核心利润

营收增长5.9%，毛利率从9.32%提升到10.06%，核心利润增速高于营收增速——这也是好信号。说明产品结构在优化，集装箱船（毛利率26.73%）占比提升拉高了整体毛利水平。

### 匹配三：核心利润与经营活动现金流量

这是最让我担心的一个匹配。净利润11.38亿，但经营活动现金流量净额是**-3.22亿**。净利润是正的，现金流是负的，而且这不是一年的事——2024年经营性现金流是-21.96亿，2025年虽然改善了18.73亿但仍然是负的。

这说明什么？说明中船防务的净利润**没有变成现金回来**。钱去哪了？

存货增加了26.22亿，预付款项增加约5亿，应收账款也有增加。也就是说，赚的钱变成了船台上的存货和客户欠的应收款。

**我给中船防务的一个核心建议是：三个匹配中，利润和现金流的匹配是最薄弱的一环。** 将来等这些存货变成船交付出去、应收款收回来，现金流会改善——但前提是交付顺利、客户按时付款。

## 五、看ROE和回报质量

2025年加权平均ROE为5.76%，比2024年的2.20%提高了3.56个百分点。投入资本回报率（ROIC）3.25%。

5.76%的ROE算高还是低？实话说，不高。如果不考虑2025年的特殊增长，中船防务过去十年的中位数ROIC只有0.87%，最惨的2018年ROIC是-7.55%。

但我认为，中船防务不应该只看一年的ROE。为什么呢？因为造船行业是典型的长周期行业——从接单到交付2-3年，ROE是波动的。2023年净利润只有4800万，2024年3.77亿，2025年10.08亿——这条曲线是向上的，但斜率要看订单交付节奏。

2025年末手持订单606亿，相当于3倍的2025年营收。也就是说，未来2-3年的饭已经端到桌上了。关键问题是：**毛利率能维持在什么水平？**

从2025年数据看，集装箱船毛利率26.73%，这是非常好的水平。但特种船（军船为主）毛利率只有2.76%，拉低了整体。如果未来高毛利的集装箱船占比继续提升，ROE突破10%是可以期待的。

## 六、看现金流：三张报表的"日子"

| 现金流项目 | 2025年 | 2024年 | 变化 |
|---|---|---|---|
| 经营性现金流 | -3.22亿 | -21.96亿 | +18.73亿 |
| 投资性现金流 | 24.91亿 | -4.28亿 | +29.19亿 |
| 筹资性现金流 | -7.78亿 | 8.07亿 | -15.85亿 |

**经营现金流-3.22亿**，虽然比2024年大幅度改善，但连续两年为负，说明造船行业"垫资生产"的特征依然明显。造船是订单式的，接了单就要采购钢板、支付人工，但船款的支付是按节点（开工、下水、交付）分批进行的——在建工程越多，占用的现金越多。

**投资现金流24.91亿**，由负转正——说明公司在收回投资或者处置资产。对于造船企业处于周期上行阶段来说，这个方向是对的：在行业景气时回收资金，而不是盲目扩张产能。

**筹资现金流-7.78亿**，说明公司在还债。结合短期借款暴增327%（从10.3亿到44.1亿）、长期借款减少45%（从约40亿到约22亿），可以看出中船防务在主动调整债务结构——把长期借款置换为短期借款。这个操作的好处是降低了利息成本（长期利率高），但代价是流动性压力增大。

用我的"面子、日子、底子"来总结：

- **面子（利润表）**：营收205亿、净利10亿——比去年好看多了，利润率在改善
- **日子（现金流量表）**：经营现金流仍然为负——日子不算宽裕，还在垫资生产阶段
- **底子（资产负债表）**：560亿总资产、217亿净资产——底子还算厚，但存货占比偏高

## 七、看风险：三个信号

### 信号一：利润与现金流的背离

净利润11.38亿 vs 经营现金流-3.22亿。这不是小事。连续两年"盈利不赚钱"，需要追问——这些利润什么时候能变成现金？如果客户延迟付款或者市场变化，利润可能就变成了坏账。

### 信号二：存货大幅增长

存货85.28亿，占净资产47%，同比增长44%。存货跌价准备只有1.94亿，计提比例2.22%。大家想一想，如果航运市场突然降温，船东要求改型或者延迟接船，这些钢板的可变现净值会怎样？计提比例是否充分？需要持续关注。

### 信号三：短期借款激增

短期借款从10亿跳到44亿，增长了327%。用短期资金支持造船这种长周期业务，存在期限错配风险。虽然造船行业景气度好、订单充足，但如果融资环境收紧，续贷压力会很大。

## 八、总结：中船防务值不值得关注？

**好的方面：**

第一，**赛道好**。全球造船行业进入上行周期，中国造船业三大指标连续16年全球第一，中船防务作为华南地区最大造船基地，直接受益。

第二，**订单充裕**。手持订单606亿，是2025年营收的3倍。排产已到2028年，未来3-4年的收入能见度很高。

第三，**产品结构优化**。毛利率26.73%的集装箱船占比在提升，这是拉高整体盈利能力的核心驱动力。

第四，**军工护城河**。军用舰船、公务船领域，中船防务有150年的军工史，这个地位不是新进入者能挑战的。

**需要谨慎的方面：**

第一，**ROE仍然偏低**。5.76%的ROE，在A股市场不算有吸引力。虽然趋势向好，但投资者需要耐心等待毛利率持续改善。

第二，**现金流问题是核心矛盾**。经营现金流连续两年为负，说明这个生意本身"垫资"特征很强。什么时候现金流转正，什么时候才是真正健康的信号。

第三，**控股型架构增加分析难度**。母公司是投资管理平台，子公司在造船——利润分配、税务筹划、资金归集都会影响合并报表的呈现。分析中船防务，必须看透两层。

第四，**存货估值风险**。85亿存货如果遇到行业拐点，减值风险不小。2.22%的计提比例是否充分，需要对照同行来看。

**一句话总结**：中船防务是一个"正在爬坡"的企业——订单充足、毛利率改善、行业景气向上，这些都写着"未来更好"；但经营现金流为负、ROE偏低、存货高企，又提醒我们"爬坡还没爬到顶"。用我的话说，**它的"底子"（资产负债表）和"面子"（利润表）都在改善，但"日子"（现金流量表）还在咬牙坚持的阶段。** 能不能在行业景气窗口期内把经营现金流转正，是我判断它能否真正走出周期底部、持续创造价值的关键指标。
