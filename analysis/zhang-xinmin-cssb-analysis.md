# 张新民视角：中国船舶（600150）——合同负债视角下的财务质量分析

> "流动和非流动没有战略，但经营和投资有战略。"
>
> "天天靠输血的企业很难有底气，造血能力极强、不靠输血的企业底气就是强。"

---

## 一、基本画像

中国船舶，全名中国船舶工业股份有限公司，是中国船舶集团旗下的核心上市平台。2025年完成与中国重工的合并重组后，总资产从1,820亿跃升至4,205亿，成为全球最大的上市造船企业。

它造的船包括：集装箱船（运货物的）、LNG船（运天然气的）、油轮（运石油的）、军舰。客户是全球的航运公司和各国海军。

在我的框架里，给它定性并不容易——它不是纯粹的"经营主导型"，也不是"投资主导型"。它是一家**经营+投资并重型**企业——自己有船厂直接造船（经营资产），也控股大量子公司来整合产能（投资资产）。2025年合并中国重工后，投资属性进一步强化。

---

## 二、资产负债表重构——这才是看战略的正确方式

我先从资产负债表重构开始。这不是传统的"流动资产/非流动资产"分类，而是**经营资产/投资资产**和**经营负债/金融负债/股东入资/利润积累**的分类。流动和非流动没有战略，但经营和投资有战略。

### 左边：钱去哪了？（2025年报，总资产4,205亿）

基于2025年12月31日数据：

```
经营资产：
  货币资金                1,468亿    ← 其中含船东预付款资金，部分受限
  应收票据+账款            86亿
  预付款项                384亿      （买钢材、设备预付）
  存货                    749亿      （在建船舶+原材料）
  长期应收款              30亿
  固定资产                543亿      （船坞、设备、厂房）
  在建工程                323亿      （扩建中的船坞）
  无形资产                96亿
  其他经营资产            202亿
  经营资产合计           ≈3,881亿    ← 占比92%

投资资产：
  长期股权投资            369亿      
  其他非流动金融资产       11亿
  商誉                    1.4亿
  投资资产合计           ≈381亿      ← 占比9%

总资产                  4,262亿
```
> 注：数字可能因四舍五入和部分科目归类有微小差异。本文用的是2025年报数据。

经营资产占92%，投资资产占9%——定性为"经营主导型"。但9%的投资资产对于一家制造业企业来说不算低，说明它通过控股子公司扩张的意图很明显。

### 右边：钱从哪来？

```
经营负债（不用付利息的"好负债"）：
  应付票据+账款            517亿     ← 供应商赊账
  合同负债               1,522亿     ← ★ 船东预付款——最关键的信号
  应付职工薪酬+税费        142亿
  其他应付款              163亿
  其他经营负债            242亿
  经营负债合计           ≈2,586亿    ← 占比61%

金融负债（要付利息的）：
  短期借款                76亿
  长期借款                149亿
  应付债券                32亿
  金融负债合计           ≈257亿      ← 占比6%

股东入资：
  股本+资本公积          1,150亿     ← 主要是2025年合并增发

利润积累：
  盈余公积+未分配利润      235亿
  少数股东权益             62亿
  所有者权益合计         1,496亿     ← 占比36%

总负债+权益            4,205亿
```

## 三、造血能力：中国船舶的"底气"在哪？

重构完之后，张新民的第一个判断就出来了：

**中国船舶的造血能力极强——全A股比例最高。**

为什么？看右边的结构：

经营负债2,586亿，占总资金来源的61%。其中**合同负债1,522亿**是重点——船东签约后就把20-30%的预付款打过来，船还没造、钱先到手。供应商愿意赊账517亿。这些是"不用付利息"的负债，说明中国船舶在产业链里地位极其强势。

金融负债只有257亿，占6%——对于总资产4,205亿的企业来说，这点贷款几乎可以忽略不计。它不是在靠银行活着。

**合同负债的变化趋势，揭示了业务的爆发力：**

| 时间 | 合同负债 | 同比变化 | 占总资产 |
|------|---------|---------|---------|
| 2021年 | 483亿 | — | 30.1% |
| 2022年 | 495亿 | +2.5% | 30.4% |
| 2023年 | 625亿 | +26.3% | 35.2% |
| 2024年 | 1,357亿 | +117% | 33.6% |
| 2025年 | 1,522亿 | +12.2% | 36.2% |
| 2026Q1 | 1,524亿 | 持平 | 36.4% |

从483亿增长到1,524亿，短短五年增长了215%。2024年从625亿跳到1,357亿，有一半原因是合并中国重工带来的，但另一半是实质性的订单增长。到了2025-2026年，合同负债稳定在1,500亿以上，说明订单已经充分填满了产能。

**36.4%的合同负债占比，在全A股大型企业中排名第一。** 这就叫造血能力——不是靠银行的贷款活着，是靠客户的预付款活着。

### 对比：不同的行业、不同的"造血"方式

| 企业 | 合同负债/总资产 | 行业特点 |
|------|-------------|---------|
| 中国船舶 | **36.4%** | 造船——船东预付20-30% |
| 保利发展 | 20.7% | 房地产——期房预售 |
| 中国建筑 | 10.7% | 建筑工程——工程预付款 |
| 中国石化 | 5.9% | 石油化工——油卡预充值 |
| 贵州茅台 | 2.7% | 白酒——经销商预付少量 |

中国船舶的比例是最高的，且是制造业而非房地产。这说明全球造船需求的旺盛程度。

## 四、核心利润——真相只有一个

张新民说："就连最基本的'营业利润'，都泛化到无以复加的地步了。"

为什么？因为营业利润里可以塞进投资收益、公允价值变动、资产处置收益等各种东西。所以我要用"核心利润"来穿透：

### 计算2025年核心利润

```
营业收入                        1,520亿
- 营业成本                      1,329亿
- 税金及附加                      6亿
- 销售费用                        3亿
- 管理费用                        64亿（含约32亿研发费用）
- 财务费用（利息支出部分）       约3亿（财务费用-23.35亿是净利息收入，不是支出）
≈ 核心利润                      约115亿
```

等一下。核心利润115亿，但报表净利润是105亿。这里有一个特殊情况：中国石化的财务费用是正数（付利息），但中国船舶的财务费用是**负数**（-23亿）。

财务费用是负数意味着什么？意味着**它的利息收入大于利息支出**——它存银行的钱（1,468亿现金）产生的利息，超过了它借的钱（257亿贷款）要付的利息。这就是张新民说的"底气"——你手里现金比负债多，银行利息都赚得比付的多。

### 利润拆解（2025年）

```
净利润 105亿
  ├── 核心利润       ~115亿     ← 真正的经营能力（含利息收入）
  ├── 投资收益         16亿     ← 子公司贡献
  ├── 资产减值损失     -7亿     
  └── 其他调整         -19亿    ← 所得税24亿 + 少数股东损益
```

**核心利润占主导**——这是健康的利润结构。105亿净利润是有真正的经营活动支撑的，不是靠卖资产或炒股票凑出来的。

### 核心利润率趋势

| 年份 | 核心利润率 | 净利润率 | 说明 |
|------|----------|---------|------|
| 2020 | ≈1.5% | 0.5% | 周期低谷，接近亏损 |
| 2021 | ≈1.2% | 0.4% | 低谷持续 |
| 2022 | ≈2.5% | 1.5% | 开始回升 |
| 2023 | ≈5.8% | 3.9% | 周期确认上行 |
| 2024 | ≈6.2% | 4.9% | 持续改善 |
| 2025 | ≈7.6% | 6.9% | 合并后规模效应显现 |

从0.5%到7%，核心利润率的提升反映了造船行业从低谷到繁荣的完整周期。

## 五、三个匹配测试

张新民说：固定资产、营业收入、核心利润三者必须匹配。

### 匹配1：固定资产 vs 营业收入

```
2020: 固定+在建=264亿 → 营收552亿 = 每1元资产产出2.1元收入
2023: 固定+在建=263亿 → 营收748亿 = 每1元产出2.8元收入 
2025: 固定+在建=866亿 → 营收1,520亿 = 每1元产出1.8元收入
```

**2025年有所下降**——原因是合并后并入了中国重工的固定资产，但合并首年营收协同还没有完全释放。短期看效率下降，中期要看整合效果。

### 匹配2：营业收入 vs 核心利润

```
2020: 营收552亿 → 核心利润约8亿   → 利润率1.5%
2023: 营收748亿 → 核心利润约43亿  → 利润率5.8%
2025: 营收1,520亿 → 核心利润约115亿 → 利润率7.6%
```

**高度匹配。** 营收和核心利润同步增长，且利润率持续提升——这是最理想的信号。说明规模扩张的同时盈利能力也在增强。

### 匹配3：核心利润 vs 经营活动现金流量

```
2020: 核心利润≈8亿 → OCF 9.7亿   → ✅ 匹配
2023: 核心利润≈43亿 → OCF 182亿  → ✅ OCF远超核心利润
2025: 核心利润≈115亿 → OCF 78亿  → ⚠️ 核心利润 > OCF
```

2025年出现了"背离"——核心利润115亿，但经营现金流只有78亿。为什么？

原因是**合同负债（船东预付款）**的增量在放缓。2024年合同负债增加了732亿（从625亿到1,357亿），这是巨额现金流入；2025年只增加了165亿。现金流入的节奏放缓了，而利润在持续增长。

对于造船企业来说，这不是严重问题——这是因为订单高峰期已过、进入交付高峰期。现金流会滞后利润1-2年出现。

## 六、关键风险信号

### 1. 应收账款暴增

| 年份 | 应收账款 | 占营收比 |
|------|---------|---------|
| 2020 | 50亿 | 9.1% |
| 2023 | 31亿 | 4.2% |
| 2024 | 18亿 | 2.3% |
| 2025 | **86亿** | **5.7%** |

2025年应收账款从18亿飙升到86亿，增长近5倍。虽然86亿对于1,520亿营收来说不算高，但这个增速需要警惕。合并中国重工后，部分客户的回款质量需要时间验证。

### 2. 合同资产增长

合同资产从2021年的46亿增长到2026Q1的170亿。合同资产是什么意思？就是船已经造了一部分，达到了确认收入的条件，但还没到收款节点。这个数字增长说明**在建船舶的数量和价值在增加**——这意味着未来2-3年的收入已锁定。

### 3. 存货规模

| 年份 | 存货 | 占总资产 |
|------|-----|---------|
| 2020 | 314亿 | 20.6% |
| 2023 | 363亿 | 20.4% |
| 2025 | 749亿 | 17.8% |

存货翻倍了，但占总资产比例下降——说明资产增长（合并）快于存货增长。但如果造船周期逆转，这些在建船舶的价值可能面临减值风险。

### 4. 商誉——极小，不是问题

商誉只有1.4亿，对于4,205亿的资产规模来说可以忽略不计。这说明中国船舶的扩张方式以"吸收合并"为主，没有产生巨额商誉——不像很多并购驱动的企业，商誉几十上百亿。

## 七、张新民的最终诊断

**结论：中国船舶的财务状况质量属于"上等"。造血能力是全A股最强的之一，利润结构健康、核心利润主导。风险在于周期逆转和合并在整合期产生的效率波动。**

用"面子、日子、底子"评价：

| 维度 | 评价 | 依据 |
|------|------|------|
| **面子（利润表）** | ✅ 好看 | 核心利润率7.6%，净利润105亿，五年增长40倍 |
| **日子（现金流量表）** | ⚠️ 略紧 | 经营现金流78亿，低于核心利润115亿——合同负债增量放缓是主因，不是经营恶化 |
| **底子（资产负债表）** | ✅ 很厚 | 净现金1,211亿，合同负债1,522亿——全是客户预付 |

**最核心的优势：合同负债/总资产比例36.4%，全A股大型企业最高。** 这意味着超过三分之一的资产来自于客户的预付。用张新民的话说："天天靠输血的企业没底气，造血能力极强、不靠输血的企业底气就是强。"

但有一个根本性风险：造船是强周期行业。2025-2026年的合同负债高峰对应的是全球造船订单的高峰。如果新船订单开始下滑，合同负债将从"蓄水池"变成"逐渐干涸的池塘"——届时核心利润、现金流也会跟着下滑。

**我的看法是：中国船舶的财务状况本身非常健康。但投资价值的判断，取决于你对全球造船周期位置的判断——这不在财务分析的范畴内，需要你对行业有自己的理解。**
