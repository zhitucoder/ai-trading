# 中国船舶（600150）百万资金作战计划

> 底仓等主升 + 融资做T降成本 | 总资金100万，融资额度50万

---

## 一、资金架构

| 项目 | 金额 | 说明 |
|------|------|------|
| **总资金** | 100万 | 自有资金，满仓打底 |
| **融资额度** | 50万 | 券商授信，专款专用 |
| **底仓（自有）** | 100万 | 长期持有，等主升浪 |
| **T仓（融资）** | 50万 | 震荡期做T，降底仓成本 |
| **总敞口上限** | 150万 | 底仓100万+T仓50万 |

### 两套资金各司其职

```
┌─────────────────────────────────────────────┐
│  底仓（100万自有资金）                        │
│  ┌─────────────────────────────────────────┐ │
│  │ 任务：死拿不动，等主升浪到42以上          │ │

│  │ 手法：35↓买→36↑卖，单笔吃2~3%           │ │
│  │ 禁则：不隔夜、不追高、不恋战            │ │
│  └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

## 二、底仓策略：100万死拿等主升

### 建仓方式

当前价位若尚未建仓，分批买入：

| 价格区间 | 买入金额 | 累计 | 说明 |
|----------|---------|------|------|
| 37~38元 | 20万 | 20万 | 当前价附近，先打底 |
| 34~36元 | 40万 | 60万 | 震荡中轴下方，加仓 |
| 30~33元 | 40万 | 100万 | 箱体下沿，满仓 |

若已满仓（如用户当前状态），**底仓不动**，直接进入下一阶段。

### 底仓退出规则

| 条件 | 操作 | 逻辑 |
|------|------|------|
| **主升浪确认**：放量突破42且连续3天站稳 | 继续持有 | 主升浪刚启动 |
| **从高点回撤15%以上** | 减半仓 | 保护利润 |
| **基本面恶化**（订单大幅下滑、亏损）| 全部清仓 | 不再恋战 |
| **其他情况** | 不动 | 震荡期只做T不减仓 |

## 三、T仓策略：50万融资反复做T

### 时间窗口

```
9:30 ─────────────── 开盘不操作，等方向
10:00~11:00 ──────── 上午低点窗口，适合买入
11:00~14:00 ──────── 横盘观望
14:00~15:00 ──────── 尾盘定方向，适合卖出
```

**最佳做T时段**：10:00~10:30（日内低点常见于此）+ 14:30~15:00（尾盘拉高）

### 单笔做T模板

| 步骤 | 操作 | 仓位 | 条件 |
|------|------|------|------|
| ① 买入 | 融资买入 | 5~10万（可用额度的10~20%） | 盘中跌到**35.x元** |
| ② 卖出 | 卖出同等数量底仓 | T+0平仓 | 拉起到**36.x元** |
| ③ 记账 | 差价利润计入"成本降低池" | — | 累计满10万可还融资额度 |
| ④ 循环 | 等待下一次35↓ | — | 至少间隔1个交易日 |

### 为什么卖底仓、不卖融资仓

```
融资买入的股不能当天卖（T+1）
底仓的股可以当天卖（已有持仓）
所以：融资买入→卖出底仓（T+0完成）
```

### 单笔收益核算

| 项目 | 数值 |
|------|------|
| 买入价 | 35.0元 |
| 卖出价 | 36.0元 |
| 买入金额 | 10万元 |
| 买入股数 | 2,800股（向下取整至100股倍数） |
| 毛利 | 2,800 × 1.0 = **2,800元** |
| 印花税(0.05%) | ~18元 |
| 佣金(0.025%) | ~9元 |
| 净利 | **~2,770元** |
| 单次收益率 | **2.77%** |

### 执行纪律

| 规则 | 原因 |
|------|------|
| ✅ 到价必须出，不贪多 | 36.0到了就卖，不等36.5 |
| ✅ 一天只做一轮 | 多做容易做飞 |
| ✅ 没跌到35不进场 | 主动买=追高 |
| ❌ 永远不隔夜 | 融资T仓当天了结 |
| ❌ 亏损不补仓 | 35买了跌到34？认亏，不补 |

## 四、成本降低测算

### 年度预期

| 假设 | 保守 | 中性 | 乐观 |
|------|------|------|------|
| 月均做T次数 | 8次 | 12次 | 16次 |
| 单笔净利 | 2,500元 | 2,700元 | 3,000元 |
| 月度降低 | 20,000元 | 32,400元 | 48,000元 |
| **年度降低** | **24万** | **38.9万** | **57.6万** |
| 底仓100万成本降幅 | **24%** | **38.9%** | **57.6%** |

即使保守估计，一年也能把成本从37元降到**28元附近**。主升浪一来，利润空间极大。

## 五、不同阶段的切换策略

### 阶段一：震荡期（34~38元）

| 动作 | 底仓 | T仓 |
|------|------|-----|
| 操作 | 不动 | 正常做T |
| 频率 | — | 每周2~3次 |
| 目标 | 等 | 降成本 |

### 阶段二：突破初期（38~40元）

| 动作 | 底仓 | T仓 |
|------|------|-----|
| 操作 | 不动 | **T仓减半，停止做T** |
| 逻辑 | 等确认 | 先锁定一半T利润 |
| 观察 | 成交量是否放大 | 是否站稳40 |

### 阶段三：主升浪确认（站稳42+放量）

| 动作 | 底仓 | T仓 |
|------|------|-----|
| 操作 | 死拿 | **全部清仓，不再做T** |
| 逻辑 | 等目标位 | 50万利润已落袋 |
| 风险 | 防止T飞 | 主升浪不做T |

### 阶段四：见顶信号

| 出现以下任意信号 | 操作 |
|----------------|------|
| 从高点回撤15% | 底仓减半 |
| 连续3日缩量滞涨 | 底仓减1/3 |

## 六、风险管理

### 保证金监控

| 指标 | 安全线 | 警戒线 | 危险线 |
|------|--------|--------|--------|
| 维持担保比例 | >400% | 300~400% | <300% |
| 融资使用率 | <60% | 60~80% | >80% |
| T仓隔夜情况 | 从不隔夜 | — | — |

### 最坏情景推演

| 情景 | 概率 | 应对 |
|------|------|------|
| 跌到30元箱体下沿 | 中 | 底仓不动，T仓用剩余额度抄底 |
| 跌破30元 | 低 | 底仓止损一半，停T |
| 直接拉涨到42不看回头 | 低 | T仓没机会，但底仓有30%+收益，不亏 |
| 持续震荡到年底 | 高 | T仓持续做，成本越降越低 |

### 三个绝对不能犯的错

1. **T仓变加仓**：35买了，跌到34不舍得卖，补仓，变成第二个底仓 → 违反纪律，直接停T一周
2. **做飞主升浪**：突破40后还在做T，涨到42卖了等回调，结果一去不回头 → 突破40立即停T
3. **融资隔夜**：当天35买，没拉起来，留着过夜 → 利息+风险双损，连续两次直接冻结T仓一个月

## 七、一句话总结

> **100万底仓死等主升浪，50万融资在34~38震荡区反复做T。震荡区降成本越狠，主升浪利润越厚。记住三不——不满仓融资、不隔夜做T、不突破40还做T。**
