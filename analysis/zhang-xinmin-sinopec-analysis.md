# 张新民视角：中国石化（600028）

> "天天靠输血的企业很难有底气，造血能力极强、不靠输血的企业底气就是强。"

---

## 一、这是一家什么企业？

中国石化，全称中国石油化工股份有限公司。它是全球最大的炼油企业和第二大化工产品生产商。2025年营收2.78万亿，资产总额2.16万亿。国有控股，国资委直属。

通俗点说：中国每两滴汽油里就有一滴是从中石化的炼油厂出来的。它的加油站遍布全国——"朝阳群众"品牌就是它。

在我这套框架里，首先要给它定性——它是典型的**经营主导型企业**。什么意思呢？它的资产主要是炼油厂、化工厂、加油站这些固定资产和经营资产，不是靠投资别的公司来赚钱的。

---

## 二、资产负债表重构

我先把资产负债表从"流动/非流动"重新分类为"经营/投资"视角，这才是看战略的正确姿势。

### 左边：钱去哪了？（2025年报总资产21,556亿）

```
经营资产：
  货币资金                1,523亿    （日常经营所需）
  应收票据+账款            517亿
  存货                    2,308亿    （原油和成品油库存）
  固定资产                7,703亿    ← 最核心的资产：炼油厂+化工厂
  在建工程                1,649亿    （还在扩建）
  无形资产                1,483亿    （油田权益+土地使用权）
  其他经营资产            4,856亿
  经营资产合计           ≈20,039亿    ← 93%是经营资产

投资资产：
  长期股权投资              1,014亿
  其他投资资产              503亿
  投资资产合计            1,517亿      ← 仅7%

总资产                  21,556亿
```

**结论：中国石化是极度纯粹的"经营主导型"企业**——93%的资产投入在经营活动中。这不是一家"投资公司"，是一家实实在在的炼油和化工企业。

## 三、造血能力评估——张新民最核心的判断

重构完后，判断就出来了：

**中国石化的造血能力极强，但正在减弱。**

怎么看的？

**有利的一面：**
- 经营负债7,186亿，占总资金来源的33%
- 其中合同负债1,264亿——客户预付的油款和化工产品定金
- 应付账款1,970亿——供应商愿意给它赊账
- 金融负债只有2,802亿，占总资产13%
- 这符合"造血能力强"的特征——不是靠银行借钱活着

**不利的一面：**
- 看趋势：利润积累虽然高达3,446亿，但2025年净利润只有352亿
- 对比2021年净利润850亿，**五年下降59%**
- 如果利润继续下滑，总有一天积累会被消耗

说白了，中国石化是一家"造血能力不错但目前正在失血"的企业。

## 四、核心利润分析——利润质量的真相

计算2025年的核心利润：

```
营业收入                    27,836亿
- 营业成本                  23,414亿
- 税金及附加                约1,820亿（石油行业特殊税费高）
- 销售费用                    593亿
- 管理费用                    554亿
- 研发费用                    148亿（含在管理费用中）
- 财务费用（利息部分）         147亿
≈ 核心利润                约1,158亿
```

等一下。核心利润1,158亿，但报表上营业利润只有405亿，净利润只有352亿。

**问题出在哪？** 中国石化的"税金及附加"极高——1,820亿。为什么？石油行业有特别收益金、资源税、消费税等，这些是炼油行业的特殊成本结构。这些税是刚性支出，不随油价波动而减少。

**换个角度看**——扣除所有税费后的核心利润率：

```
核心利润率 = 1,158亿 / 27,836亿 = 4.2%
```

对于一家制造业企业来说，4.2%的核心利润率不算高。对比一下：贵州茅台的核心利润率超过50%。这说明石油化工行业的利润空间很薄——你每卖100块钱的产品，只有4块2是真正属于自己的经营利润。

中国石化的利润结构是：**纯经营型的"价值创造"模式**。它的利润几乎全部来自经营活动，不像一些控股集团靠投资收益撑门面。这一点是健康的，但利润率偏低。

## 五、三个匹配测试——找出脱节信号

### 匹配1：固定资产 vs 营业收入

```
2021: 固定资产5,989亿 → 营收27,409亿    = 每1元资产产生4.6元营收
2025: 固定资产7,703亿 → 营收27,836亿    = 每1元资产产生3.6元营收
```

**不匹配。** 固定资产增长了29%，营收几乎没有增长。这意味着每投入1块钱买设备，产出的收入在下降。这是产能利用率下降的信号——你的炼油厂比五年前多了，但炼的油没有多多少。

### 匹配2：营业收入 vs 核心利润

```
2021: 营收27,409亿 → 核心利润约1,800亿（估计）
2025: 营收27,836亿 → 核心利润约1,158亿
```

**同样不匹配。** 营收微增，但核心利润下降了约35%。原因是成本结构变化——原油价格波动、税负增加、销售费用居高不下。

### 匹配3：核心利润 vs 经营活动现金流量

```
2025: 核心利润约1,158亿 → 经营现金流1,625亿
```

**基本匹配，但经营现金流远大于核心利润。** 为什么？主要原因是**折旧摊销**——固定资产7,703亿，每年折旧就在700亿以上。折旧是"非现金支出"，所以在利润表里扣除了，但现金没有真的流出去。

这就引出一个关键判断：

> 中国石化的经营现金流远高于净利润（1,625亿 vs 352亿），说明它的利润质量被严重低估——大量折旧掩盖了真实的现金产生能力。你看到利润表上只有352亿，但实际上经营活动中产生了1,625亿的现金。

## 六、合同负债分析

中国石化的合同负债长期稳定在1,200-1,400亿之间：

| 年份 | 合同负债 | 占总资产 |
|------|---------|---------|
| 2021 | 1,246亿 | 6.6% |
| 2022 | 1,254亿 | 6.4% |
| 2023 | 1,272亿 | 6.3% |
| 2024 | 1,277亿 | 6.1% |
| 2025 | 1,264亿 | 5.9% |

**极其稳定。** 这1,264亿主要是加油站的预充值卡余额和化工产品的预付款。它是中国石化"现金蓄水池"的一部分——客户先付钱、后消费。这个蓄水池始终维持在1,200亿以上，说明公司的市场终端触达能力非常稳固。

横向对比：中国船舶的合同负债占资产36.4%（船东预付款暴增），中国建筑10.7%，中国石化5.9%。5.9%不算高，但胜在极其稳定——不像造船行业那样大起大落。

## 七、风险与隐患

1. **利润五年持续下滑——最大的问题**
   - 2021年净利润850亿 → 2025年352亿，下降59%
   - 原因：国际油价波动、炼化产能过剩、新能源替代加速
   - 这不是周期波动——这是结构性变化

2. **固定资产效率下降**
   - 固定资产收益率从4.6倍降到3.6倍
   - 每年还在投1,600亿在建工程——钱投进去，能产生多少回报？不确定

3. **替代风险——最大的"黑天鹅"**
   - 电动车替代汽油车，对中石化的成品油业务是长期利空
   - 炼油产能过剩（中国炼油能力已经超过需求约2亿吨/年）
   - 化工业务面临民营大炼化（恒力、荣盛）的激烈竞争

4. **税费负担沉重**
   - 税金及附加1,820亿，超过净利润的5倍
   - 石油行业是国家的"提款机"——这部分是政策风险

## 八、张新民的最终诊断

**结论：中国石化的财务状况质量属于"中游偏上"，但趋势在下行。**

优势：
- ✅ 造血能力极强——经营负债7,186亿，产业链话语权稳固
- ✅ 现金流极为充沛——经营现金流1,625亿，远超账面利润
- ✅ 合同负债稳定在1,200亿以上——客户基础扎实
- ✅ 资产负债率54%，在重资产行业里算合理
- ✅ 固定资产和无形资产的折旧摊销是"真实的现金来源"

劣势：
- ❌ 利润连续五年下滑——从850亿跌到352亿
- ❌ 固定资产收益率持续下降——1块钱资产产出越来越少的收入
- ❌ 核心利润率只有4.2%——利润空间太薄，抗风险能力弱
- ❌ 新能源替代是结构性威胁，不是周期性的

**张新民不会告诉你"该不该买"。** 但他会告诉你这家企业的"底子"怎么样。

用"面子、日子、底子"评价：
- **面子（利润表）**：不好看——净利润352亿，而且还在往下走
- **日子（现金流量表）**：过得不错——经营现金流1,625亿，真金白银在进来
- **底子（资产负债表）**：厚实，但在变薄——资产2.16万亿，造血能力在减弱

中国石化像是一个家底殷实的老牌企业——账上有钱、产业链强势、不靠银行过日子，但核心业务正在被时代慢慢侵蚀。它的财务质量是"健康但不再增长"——没有倒闭风险，但也没有什么想象力。
