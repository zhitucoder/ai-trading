## 三、融资融券操作策略（涨跌都赚钱）

### 核心逻辑

既然已经**满仓**（无多余资金），利用**融资账户**在低位借钱加仓，高位卖出还钱，赚取差价。

**关键操作顺序：**

```
低位（34↓）：融资买入 → 持仓增加
高位（38↑）：卖出持仓 → 还融资款 → 锁定利润
回落（34↓）：再次融资买入 → 循环
```

### 具体价位方案

| 触发价 | 操作 | 方向 | 仓位比例 | 说明 |
|--------|------|------|---------|------|
| **≤34.0元** | 融资买入 | 做多 | 可用融资额度的30% | 第一买入点，接近MA20/MA120 |
| **≤33.0元** | 融资买入 | 做多 | 再追加30% | 进入强支撑区 |
| **≤31.5元** | 融资买入 | 做多 | 剩余40%全仓 | 接近箱体下沿，极佳买点 |
| **≥38.0元** | 卖出持仓 | 做空/止盈 | 卖出1/3融资仓 | 接近MA50，第一阻力位 |
| **≥39.5元** | 卖出持仓 | 做空/止盈 | 再卖1/3 | 强阻力区 |
| **≥41.0元** | 卖出持仓 | 做空/止盈 | 清空所有融资仓 | 接近箱体上沿，落袋为安 |

### 每笔交易预期收益

| 买入价 | 卖出价 | 单次收益率 | 说明 |
|--------|--------|-----------|------|
| 33.0元 | 38.0元 | **+15.2%** | 下沿买入→上沿卖出，最优波段 |
| 34.0元 | 38.0元 | **+11.8%** | 第一买点→第一卖点 |
| 34.0元 | 39.5元 | **+16.2%** | 买点→强阻力，最佳组合 |
| 31.5元 | 38.0元 | **+20.6%** | 极限下沿→上沿，大波段 |

> 注意：这些收益率**不包含融资利息**。目前融资利率约5~6%/年，按持仓1~2个月估算，资金成本约0.5~1%，实际操作中需扣除。

## 四、已有满仓持仓的对冲方案

如果目前持有的**全部是股票（非现金）**，无法直接融资买入（融资需要保证金）。解决方案：

### 方案A：存量转换（推荐）

```
1. 当前满仓 → 择机在38~39元卖出50%仓位
2. 获得现金 → 存入信用账户作为保证金
3. 保证金可用 → 等待34↓融资买入
4. 融资买入后 → 等待38↑卖出还券
5. 循环操作，逐步降低持仓成本
```

### 方案B：先融券卖出，后买券还券（日内T+0做空）

```
1. 在38元上方  → 融券卖出（借券做空）
2. 在34元下方  → 买券还券（平仓空单）
3. 同时用原有持仓 → 在低位补仓
4. 高位再卖出→还融资
```

### 方案C：底仓不动，仅用融资额度做波段

```
1. 保持当前满仓不动
2. 信用账户中还有融资额度可用
3. 34↓融资买入 → 38↑卖出现有持仓（不卖融资仓）
4. 用卖出的钱还融资款
5. 效果：底仓不变，额外赚取波段差价
```

## 五、风险控制

### 止损规则

| 情景 | 止损线 | 操作 |
|------|--------|------|
| 融资买入后跌破30元 | 跌破箱体下沿 | 无条件止损，减少融资仓位50% |
| 连续3日收盘低于32元 | 趋势已坏 | 清空所有融资仓位 |
| 融资买入后单日跌超5% | 异常波动 | 立即减仓1/2 |

### 仓位管理红线

- **融资总仓位不超过总资产的50%**（券商上限通常为100%，但保守操作不应超过50%）
- **单笔融资不超过可用额度的40%**
- **两次融资买入间隔至少3个交易日**（避免抄底抄在半山腰）
- **账户维持担保比例始终>300%**（低于300%会触发追加保证金）

### 融资成本控制

| 持有时间 | 融资成本估算（年化5.5%） | 对收益率影响 |
|----------|------------------------|------------|
| 1个月 | 0.46% | 可控 |
| 2个月 | 0.92% | 需考虑 |
| 3个月以上 | >1.38% | 不建议长期持有融资仓 |

**建议：每笔融资仓持有时间控制在2个月以内。**
