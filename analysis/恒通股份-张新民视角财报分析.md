# 恒通股份（603223）财报分析

> 分析视角：张新民 · 战略视角财务报表分析框架
> 数据截止：2026年一季报

## 一、企业画像

恒通股份，所属行业：运输服务（公路运输），是一家正处于战略转型期的物流企业。

过去三年，这家企业完成了一次根本性的战略转身——从一家低毛利、高周转的LNG贸易商，转型为以港口为核心的基础设施运营商。**方向是好的，但落地是考验。**

---

## 二、资产负债表重构

### 左边：资产结构（2026-03-31）

| 项目 | 金额（亿） | 占比 |
|------|-----------|------|
| 货币资金 | 10.88 | 20.4% |
| 交易性金融资产 | 1.00 | 1.9% |
| 应收账款 | 1.53 | 2.9% |
| 存货 | 4.26 | 8.0% |
| **流动资产合计** | **20.51** | **38.4%** |
| 固定资产 | 25.09 | 47.0% |
| 无形资产 | 3.17 | 5.9% |
| **非流动资产合计** | **32.83** | **61.6%** |
| **总资产** | **53.34** | **100%** |

固定资产占比47%，这是典型的经营主导型企业。钱主要砸在了实体资产上。

**关键变化**：2022年6月时固定资产才5.2亿，到2025年6月飙升至25亿。两年翻五倍，说明企业在进行大规模的战略性基础设施投资。

### 右边：资本结构（2026-03-31）

| 项目 | 金额（亿） | 占比 |
|------|-----------|------|
| 应付账款（经营负债） | 2.30 | 4.3% |
| 短期借款 | 0.20 | 0.4% |
| 长期借款 | 5.69 | 10.7% |
| **总负债** | **11.94** | **22.4%** |
| 股本 | 7.06 | 13.2% |
| 资本公积 | 23.42 | 43.9% |
| 盈余公积+未分配利润 | 10.34 | 19.4% |
| **总权益** | **41.40** | **77.6%** |
| **负债+权益** | **53.34** | **100%** |

资产负债率22.4%，极低。但更关键的是看右边的结构。

**拆开来看**：
- **造血来源**：经营负债2.3亿 + 利润积累10.34亿 = **12.64亿（23.7%）**
- **输血来源**：金融负债5.89亿 + 股东入资30.48亿 = **36.37亿（68.2%）**

资本公积23.42亿，超过股本的3倍。说明历史上进行过大额股权融资。这家企业是靠"输血"建立了家底，现在正在向自我"造血"转型。

**有意思的矛盾**：账上货币资金10.88亿，而短期借款只有0.2亿，长期借款5.69亿。这意味着企业完全可以用自有资金偿还全部借款（现金10.88亿 > 全部借款5.89亿）。为什么不还？要么是在预留发展资金，要么是借款有特殊用途。

## 三、利润质量分析

### 利润表概览

| 指标 | 2024全年 | 2025全年 | 2026Q1 |
|------|---------|---------|--------|
| 营业收入 | 20.02亿 | 13.61亿 | 3.38亿 |
| 营业成本 | 17.22亿 | 9.46亿 | 2.10亿 |
| 毛利率 | 14.0% | **30.5%** | **37.7%** |
| 营业利润 | 1.79亿 | 2.97亿 | 1.03亿 |
| 净利润 | 1.54亿 | 2.66亿 | 0.95亿 |
| 净利率 | 7.7% | 19.5% | 28.2% |

### 核心利润拆解（2025全年）

| 项目 | 金额（亿） |
|------|-----------|
| 营业收入 | 13.61 |
| 减：营业成本 | 9.46 |
| 减：销售费用 | 0.29 |
| 减：管理费用 | 0.80 |
| 减：财务费用 | 0.18 |
| **约等于：核心利润** | **2.88** |
| 加/减：投资收益 | -0.08 |
| 加：营业外收入 | 0.02 |
| **营业利润** | **2.97** |
| **净利润** | **2.66** |

核心利润占营业利润的97%。利润结构非常干净——利润是经营做出来的，不是靠变卖资产或会计调整。

**三个匹配检查**：

1. **固定资产与营收**：25亿固定资产对应13.61亿营收，周转率仅0.54。偏低。但港口是基础设施，一旦建好要用很多年，不能期望投产第一年就满负荷运转。裕龙石化一期2024年9月才投产，产能爬坡需要时间。

2. **营收与核心利润**：毛利率从2021年的3.4%提升到2025年的30.5%。营收在降但毛利率在升——这说明企业不是在做"收缩"，而是在做"换挡"。

3. **核心利润与经营现金流**：2026Q1核心利润约1.08亿，但经营现金流为**-1.3亿**。利润和现金流出现了明显的背离。一季度虽然有季节性因素，但这么大的缺口需要仔细排查——是应收账款激增？还是存货积压？如果连续两个季度经营现金流为负，就要重新审视利润质量。

### 营收大幅下滑的真相

2024年营收20.02亿，相比2023年的37.56亿下降46.7%；2025年进一步降至13.61亿。表面看很吓人。

但原因是主动的战略调整——LNG业务从重资产"运贸一体"转向轻资产"承运"模式，主动压缩贸易规模。公司2024年下半年开始处置约1/3的LNG车辆，大幅减少运贸一体业务。

结果是：营收少了，但利润从1.54亿增长到2.66亿，增长73%。**这是主动瘦身，不是被动衰退。**

## 四、业务结构分析

2025年公司已形成三大板块：

| 业务板块 | 收入（亿） | 占比 | 毛利率 |
|---------|-----------|------|--------|
| 港口业务 | 5.76 | 42.4% | 43.2% |
| 货物运输 | 4.16 | 30.6% | — |
| LNG贸易 | 2.79 | 20.5% | 提升6.98pct |
| 其他 | 0.90 | 6.6% | — |

**港口业务**已成为第一大收入和利润来源，依托裕龙石化产业园，提供船舶停泊、货物装卸服务。这是未来增长的引擎。

**LNG业务**从"运贸一体"转型为"承运为主、贸易为辅"，规避了大宗商品价格波动风险，盈利稳定性增强。

## 五、历史趋势：从70亿到13亿的转型

回顾历史，更能看清这家企业：

- **2017-2021年**：营收40-70亿，但毛利率仅2-5%。典型的大宗商品贸易商模式——量大但利薄，利润受价格波动影响极大。
- **2022年**：总资产从19.8亿跃升至45.5亿，资本公积大幅增加。重大股权融资事件发生。
- **2023-2024年**：开始战略收缩LNG贸易，投资港口基础设施。营收逐步下降，毛利率开始回升。
- **2025年**：转型基本完成。港口业务成为核心，营收稳定在13-14亿，毛利率达到30.5%。

**这是一次从"走量"到"走质"的彻底转身。**

## 六、风险信号

1. **经营现金流与利润背离**：2026Q1净利润9519万，经营现金流-1.3亿。需要持续关注是季节性因素还是趋势性恶化。如果连续两个季度经营现金流为负，就要重新审视利润质量。

2. **固定资产折旧压力**：25亿固定资产，按10年折旧估算，每年折旧约2.5亿。2025年核心利润才2.88亿，折旧吃掉了一大块。如果未来营收不能增长，折旧将严重侵蚀利润。

3. **客户集中度风险**：港口业务高度依赖裕龙石化产业园。裕龙石化一期虽然已投产，但二期计划2026年底才能投产。中间的空窗期，港口产能利用率能否持续提升？

4. **货币资金使用效率**：账上10.88亿现金，交易性金融资产1亿，合计近12亿。占总资产的22%。现金占比偏高，资金使用效率有待提高。

5. **商誉为零**：这是好消息。没有并购历史，没有商誉减值的"定时炸弹"。

---

## 七、综合评价

**优势**：
- 战略方向清晰：从低毛利贸易向高毛利基础设施运营转型
- 财务极为保守：22.4%的负债率，无商誉，利润结构干净
- 港口业务壁垒高：依托裕龙石化产业园，基础设施一旦建成，竞争壁垒天然形成
- 利润质量扎实：核心利润占比97%，几乎没有"水分"

**关注点**：
- 固定资产25亿的回报率能否持续提升
- 经营现金流能否回正
- 港口产能利用率能否持续爬坡
- 股权融资历史导致的股本稀释效应

**核心矛盾**：这家企业比三年前健康得多。原来的问题是"能不能活"，靠的是2-3%的微利在70亿营收规模上勉强运转。现在的问题是"固定资产能不能产生预期的回报"，靠的是25亿基础设施能否持续满负荷运转。

方向对了，落地是考验。我建议每季度跟踪两个指标：经营现金流和港口业务收入增速。
