# 彼得·林奇视角：中国船舶（600150）分析

> "Know what you own, and know why you own it."

## 第一步：这公司是做什么的？

**中国船舶**是中国最大的国有造船企业，也是全球最大的造船集团。它造的是真正的"大东西"——集装箱船、油轮、LNG船、军舰。它的客户是全球的航运公司和各国海军。

通俗点说：全球贸易的货物有80%通过海运，这些船大多数是中国造的，而中国船舶是其中最核心的那个玩家。

## 第二步：它属于哪一类？（Six Categories Test）

这是林奇框架里最关键的一步——**不同类型的股票有不同的操作规则，你不能用同样的方法对待它们。**

### 排除法

| 类型 | 符合？ | 理由 |
|------|--------|------|
| **Slow Grower（缓慢增长型）** | ❌ | 营收增长93%，哪来的"缓慢" |
| **Stalwart（稳健增长型）** | ❌ | 波动太大，周期太强，不是可口可乐那种稳定增长 |
| **Fast Grower（快速增长型）** | ⚠️部分 | 93%的增长像是Fast Grower，但问题在于：这不可持续。造船是周期行业，不是长期20-25%增长的消费品牌 |
| **Cyclical（周期型）** | ✅ **核心类别** | 造船是全球最经典的周期行业之一——订单跟着全球贸易周期走，一艘船从定到交要2-4年，供需匹配极度困难 |
| **Turnaround（困境反转型）** | ✅ **有历史** | 2016年亏损30亿，到2025年盈利105亿，这是一个教科书式的困境反转 |
| **Asset Play（资产隐藏型）** | ✅ **有内容** | 账面净现金1,243亿，市值约2,586亿——现金占了股价的48% |

### 最终判定：周期型（Cyclical），附带资产隐藏型属性和困境反转历史

**为什么不是Turnaround？** 困境反转型的最佳买入时机是公司还在挣扎、华尔街已经放弃的时候。中国船舶的困境反转已经在2016-2023年完成了。现在它处在周期的高歌猛进阶段，这已经不是"反转"——这是"繁荣"。

**为什么不是Fast Grower？** 造船行业的繁荣无法永远持续。历史上造船周期通常持续3-5年，然后产能过剩，价格崩溃。把93%的增长当作长期增长率是最大的错误。

### 林奇的周期股黄金法则

**对于周期股，P/E越低越危险，P/E越高越安全。**

为什么？因为周期股在利润高峰时P/E最低（分母最大），但那正是周期的顶点——接下来就是下滑。在利润低谷时P/E最高甚至为负，但那才是买入时机。

中国船舶的当前数据：

| 指标 | 数值 |
|------|------|
| 当前股价 | ¥34.36 |
| TTM净利润 | ¥229.8亿 |
| 市值 | ¥2,586亿 |
| TTM P/E | **~11.3倍** |
| 2025年P/E | **~24.6倍**（因为合并是一次性事件，2025年数据不完全可比） |
| 剔除现金后P/E | ~5.9倍（市值2,586 - 净现金1,243 = 1,343亿 / 229.8亿） |

当前P/E 11.3倍，对于一个年增长93%的公司来说看起来极其便宜。但等一下——**这就是周期股的陷阱。**

如果我用2025年正常化后的利润（约50-60亿，不含合并一次性影响），P/E大约是43-52倍。如果考虑巅峰已过、利润开始下滑，实际P/E可能比看起来高得多。

### 现金——真正的安全垫

| 项目 | 金额 |
|------|------|
| 账面现金 | ¥1,468亿 |
| 总负债 | ¥2,709亿（主要是预收款/合同负债——船东的预付款） |
| 有息债务 | ¥225亿 |
| **净现金** | **¥1,243亿** |
| 市值 | ¥2,586亿 |
| **净现金/市值** | **48%** |

**这意味着什么？** 你花¥100买中国船舶的股票，其中¥48是直接买到了现金。实际你为"造船业务"只支付了¥52。对于一个全球第一的造船巨头来说，这个估值不算贵——**前提是你能接受周期的下行阶段不会把你吓跑。**

### 存货/应收账款——需要警惕的信号

| 年份 | 存货 | 应收账款 | 说明 |
|------|------|---------|------|
| 2023 | ¥363亿 | ¥14亿 | 正常 |
| 2024 | ¥358亿 | ¥18亿 | 正常 |
| 2025 | ¥749亿 | ¥86亿 | **暴涨**——但主要是合并中国重工带来的，并非经营恶化 |
| 2026Q1 | ¥700亿 | ¥111亿 | 应收继续增长——需关注回款 |

林奇会特别关注这个趋势：合并之后的应收账款从¥18亿涨到¥111亿，增长6倍，远超营收增速。**这需要密切跟踪。** 如果回款恶化，就是周期即将见顶的信号之一。

### 现金流质量——造船业的特殊模式

```
2020: OCF -44.5亿（亏损年份）
2021: OCF -29.8亿
2022: OCF -0.4亿
2023: OCF +182.1亿（周期高峰确认）
2024: OCF +52.4亿
2025: OCF +77.7亿
```

造船业的特点是：船东分期付款（签约付20%，开工付20%，下水付20%，交付付40%），所以现金流经常领先利润——2023年OCF远高于净利润，2024-2025年相反。这不是经营恶化，是合同节奏问题。

## 第六步：林奇的最终判断

### 这是一个什么机会？

**这是一个周期繁荣期的资产隐蔽型机会，不是困境反转。**

如果你想按照林奇的方法操作中国船舶，你需要清楚地知道：

1. **它不是困境反转型（Turnaround）**——困境反转已经完成了。真正抓住反转的投资者是在2016-2019年买入的，那时公司在亏损、华尔街在躲避。

2. **它是周期型（Cyclical），按照周期股的操作规则：**
   - 不要用"增长率"来线性外推——93%的增长不可持续
   - 关注领先指标——波罗的海干散货指数、新船订单价格、二手船价格
   - 存货和应收账款是最重要的"预警信号"——一旦开始恶化，就该走了
   - **周期股的操作是：在高P/E时买（利润低点），在低P/E时卖（利润高点）**

3. **资产隐藏型的价值：** 净现金1,243亿提供了下行的缓冲垫。即使造船业务开始下滑，股价也不太可能跌破每股¥16.5（净现金/总股本）。这给了大约51%的下行保护。

4. **关键估值数字：**
   - 剔除现金后P/E ≈ 5.9倍（用当前盈利）
   - 剔除现金后P/B ≈ 0.28倍（企业价值/扣除现金后的净资产）
   - 但如果周期反转，盈利可能下降60-80%

### 林奇式行动计划

| 情景 | 操作 |
|------|------|
| 你是长期持有者（5年以上） | 可以买，但要接受周期下行时承受30-50%回撤。净现金提供了安全垫，但无法阻止股价下跌 |
| 你是周期交易者 | 密切跟踪新船订单价格和波罗的海指数。只要订单还在增长、价格还在涨，可以持有。一旦看到连续两个季度订单下滑，立刻卖出——不管P/E看起来多便宜 |
| 你希望找困境反转 | 这不是你要的。真正的困境反转机会可能在下一轮周期低谷（预计2028-2030年），等它亏损时再来看 |
| 你被低P/E吸引 | 警惕！周期股的低P/E是最常见的陷阱。问问自己：如果利润跌回2019年的30亿水平，P/E将超过80倍，你还能安心持有吗？ |

## 总结

中国船舶当前处于"周期繁荣+资产价值支撑"的组合中。

林奇最擅长的——在别人忽视的地方找到好股票——在这里不适用。这家公司已被大量机构研究，周期繁荣已经被充分定价。它的净现金提供了下行保护，但上行空间取决于你不确定的事：全球造船周期还能持续多久。

如果让我用一句话总结：**"这是一个'好公司，但也许是错的时机'的典型例子。如果你在2019年以困境反转的逻辑买入它，你会赚很多钱。如果你现在以低P/E的吸引买入它，你买的不是一个'便宜货'——你买的是一个周期风险。"**

> "The key to making money in stocks is not to get scared out of them."
>
> —— 但前提是你知道自己买的是什么。如果你把它当作困境反转，你可能会在周期下行时被吓出来。如果你把它当作周期股，设定好exit plan，那这笔投资才有意义。
