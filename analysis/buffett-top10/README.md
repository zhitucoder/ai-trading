## 筛选标准

| 维度 | 我的要求 | 为什么 |
|------|---------|--------|
| **ROE** | 连续3年 > 15% | 好生意的核心指标：股东每投1块钱能赚多少 |
| **负债率** | < 40% | 不能在坏年景被债务压垮 |
| **毛利率** | > 20%（越高越好） | 定价权的直接证据 |
| **护城河** | 必须有可辨识的竞争优势 | 否则明年可能就不存在了 |
| **可理解性** | 普通人1分钟能说清 | 不在能力圈外冒险 |

## Top 10 榜单

---

### 第1名：贵州茅台（600519）

**核心数据（2023→2024→2025）：**
- ROE: 34.7% → 36.9% → 33.6%
- 负债率: 18.0% → 19.0% → 16.4%
- 毛利率: 92.0% → 91.9% → 91.2%
- 净利率: 52.5% → 52.3% → 50.5%
- 经营现金流: 665亿 → 925亿 → 615亿

**护城河：文化垄断**

这不是一家白酒公司。它是中国700年酒文化的载体。茅台酒只能在贵州省茅台镇酿造——离开那片土地，味道就不对了。这不是技术优势，不是专利保护，是**地理+文化+历史的三重护城河**。

**定价权测试：** ✅ 过去十年茅台持续提价，消费者不但不跑，反而囤酒。一瓶飞天茅台出厂价969元，市场零售价长期在2500元以上——终端价格是出厂价的2.5倍。这意味着它有巨大的提价空间而不损失销量。

**资本需求测试：** ✅ 2025年资本支出仅1.1亿，经营现金流615亿。赚到的钱几乎不需要投入再生产。这是典型的"印钞机"生意。

**为什么排第一：** 在A股所有公司中，最接近可口可乐1988年的状态——文化垄断品牌、极端利润空间、几乎不需要资本投入。ROE自2022年以来持续在33-37%之间，全中国最高的大市值公司之一。

**风险：** 增长放缓（2025年营收微降1.2%），规模太大难以继续高速增长。年轻人是否还喝白酒是长期隐忧。

### 第2名：山西汾酒（600809）

**核心数据（2023→2024→2025）：**
- ROE: 37.0% → 34.8% → 30.6%
- 负债率: 35.9% → 34.2% → 28.7%
- 2025年营收: 387亿 | 净利润: 123亿
- 净利率: 32.8% → 34.0% → 31.8%

**护城河：千年品牌 + 清香型白酒龙头**

汾酒是"四大名酒"之一，是清香型白酒的绝对龙头。品牌历史可追溯到南北朝时期（公元6世纪）。在白酒行业，"排他性"非常强——你喝茅台不会换成汾酒，但汾酒的消费者有极高的品牌忠诚度。

**定价权测试：** ✅ 青花汾酒系列连续多年提价。中等偏上。

**为什么排第二：** 中国白酒行业有极强的品牌护城河，汾酒是清香型第一名。ROE连续3年超30%，在品牌白酒中仅次于茅台。2025年营收增长依然为正（虽然增速放缓），比茅台稍微健康。

**风险：** ROE逐年下降（37%→35%→31%），全国化扩张遇到区域酒企阻力。增速在放缓。

### 第3名：公牛集团（603195）

**核心数据（2023→2024→2025）：**
- ROE: 26.7% → 26.9% → 24.3%
- 负债率: 26.8% → 22.5% → 25.5%
- 2025年营收: 160亿 | 净利润: 41亿
- 净利率: 24.6% → 25.4% → 25.4%

**护城河：渠道垄断**

你可能没想过，但"做插座"可以是一门极好的生意。公牛在中国插座市场的占有率超过60%。这来自于20多年建立的**渠道网络**——覆盖全国100多万个终端网点，从一线城市到最小的乡镇都有公牛的插排挂在墙上。竞争对手想要复制这个渠道网络，需要10年以上的时间和数十亿的投入。

**定价权测试：** ✅ 同样功能的插座，公牛比其他品牌贵20-30%，但消费者仍然选择它——因为公牛的可靠性和安全形象是几十年积累的。你不敢把便宜插座插在几千块的电器上。

**为什么排第三：** 公牛是典型的"不起眼但很赚钱"的生意。ROE连续3年稳定在25%上下，波动极小。负债率始终在25%左右，财务极其保守。这是一台不需要操心就能持续赚钱的机器。

**风险：** 房地产市场下行影响装修需求。智能家居时代可能改变插座的使用场景。

---

### 第4名：泸州老窖（000568）

**核心数据（2023→2024→2025）：**
- ROE: 32.0% → 28.4% → 21.8%
- 负债率: 34.4% → 30.5% → 23.0%
- 2025年营收: 257亿 | 净利润: 109亿
- 净利率: 44.0% → 43.3% → 42.2%

**护城河："国窖1573"品牌 + 浓香型白酒标准制定者**

泸州老窖是浓香型白酒的鼻祖。"国窖1573"是其高端品牌，在千元价格带与五粮液正面竞争。拥有始建于公元1573年的国宝级窖池群，这是不可复制的稀缺资源。白酒的窖池越老，酿出的酒越好——这是一个随着时间推移护城河自动加宽的生意。

**定价权测试：** ✅ 国窖1573在千元价格带建立了稳固的品牌认知，提价能力仅次于茅台。

**风险：** ROE从2023年的32%下降到2025年的21.8%，趋势令人担忧。在行业增速放缓时，高端白酒的竞争加剧，国窖1573受到五粮液的强力压制。

### 第5名：分众传媒（002027）

**核心数据（2023→2024→2025）：**
- ROE: 26.6% → 29.3% → 20.0%
- 负债率: 25.9% → 24.0% → 27.7%
- 2025年营收: 128亿 | 净利润: 29亿
- 经营现金流: 约45亿（远超净利润）

**护城河：物理空间垄断**

分众传媒占据了中国城市主流写字楼和住宅小区的电梯广告位。这不是一种技术护城河——它是**物理空间租赁合同的护城河**。全国80%的核心电梯广告位被分众签下，竞争对手即使有钱，也没有好的电梯位可以签了。

**定价权测试：** ⚠️ 有限。广告价格在经济下行时会被压缩。但分众对品牌广告主来说是"必须投"的渠道，这使得它在经济好的时候有定价权。

**为什么排第五：** 分众在好的年景ROE接近30%，资本需求极低（主要是维护租赁合同）。它赚的是真金白银——经营现金流常年高于净利润。对一个广告公司来说，这种商业模式是顶级水平。

**风险：** 经济周期性极强（2025年ROE降到20%）。受到互联网广告和短视频的竞争威胁。电梯广告的长期存在形式可能被数字化屏幕取代。

### 第6名：福耀玻璃（600660）

**核心数据（2024→2025）：**
- ROE: 21.0% → 24.8%
- 负债率: 43.6% → 46.4%
- 2025年营收: 458亿 | 净利润: 93亿
- 净利率: 19.1% → 20.4%

**护城河：全球规模成本优势 + 曹德旺管理**

福耀是全球最大的汽车玻璃制造商，全球市场份额超过30%，中国超过65%。汽车玻璃行业门槛极高——整车厂的认证周期长达3-5年，且一旦进入供应链，更换供应商的成本极高。这是一个**高转换成本的护城河**。

福耀的另一个优势是创始人曹德旺。他是一个极其专注的实业家，40年只做玻璃。他亲自管理成本、推动自动化，使得福耀的利润率远高于竞争对手（旭硝子、圣戈班等）。

**定价权测试：** ⚠️ 对整车厂的议价能力有限。但福耀通过成本控制来维持利润率。

**为什么排第六：** 福耀在一个"不太好的行业"（汽车零部件）中做成了极好的生意。关键是它的规模——全球最大汽车玻璃制造商，成本比竞争对手低20%以上。这是**成本优势护城河**的典范。ROE在改善（21%→25%），趋势向好。

**风险：** 负债率偏高（46%）。汽车行业电动化可能改变玻璃需求结构。创始人曹德旺已高龄，接班人问题不确定。

### 第7名：亿联网络（300628）

**核心数据（2023→2024→2025）：**
- ROE: 24.7% → 29.6% → 27.9%
- 负债率: 8.6% → 8.5% → 12.8%
- 2025年营收: 60亿 | 净利润: 26亿
- 净利率: 46.2% → 47.1% → 43.1%

**护城河：全球细分市场冠军**

亿联网络做的是企业统一通信终端——你知道思科的IP电话吗？亿联就是在这个细分市场做到了全球第一。它的SIP桌面终端在全球市场份额约35%，是绝对的全球领导者。它90%以上的收入来自海外，客户包括微软Teams、Zoom等平台的生态合作伙伴。

这是一家"闷声发财"的中国公司——不做手机，不做电脑，只做企业通信的"话筒和摄像头"，却在全球做到了第一。产品毛利率稳定在65%以上。

**定价权测试：** ✅ 作为细分市场的全球领导者，亿联对中小企业客户有较强的定价权。企业级通信设备的客户黏性高，替换成本大。

**为什么排第七：** ROE连续3年稳定在25-30%，负债率极低（8-13%），这是在A股极其罕见的组合。净利率43-47%，说明它在一个不错的利基市场有着稳固的竞争地位。财务极度保守——账上现金远多于负债。

**风险：** 市场较小（营收60亿），增长天花板明显。主要依赖海外市场和微软Teams生态，有一定政策和技术标准风险。

### 第8名：双汇发展（000895）

**核心数据（2023→2024→2025）：**
- ROE: 24.1% → 23.7% → 23.9%
- 负债率: 42.3% → 42.8% → 45.3%
- 2025年营收: 593亿 | 净利润: 52亿
- 净利率: 8.5% → 8.5% → 8.7%

**护城河：规模 + 渠道 + 品牌**

双汇是中国最大的肉类加工企业。你可能觉得"杀猪卖肉"不是什么好生意——但双汇把它做成了ROE连续多年超过20%的优质生意。关键是**规模**：双汇在全国有数百万个零售终端，冷链物流网络是竞争对手无法复制的。别人卖1斤肉赚1块钱，双汇买进整猪、分割包装后每斤能赚2块钱——这就是规模经济的威力。

**定价权测试：** ❌ 定价权弱。肉价由市场决定，双汇是价格接受者。但它在加工环节有成本优势。

**为什么排第八：** 双汇是A股少有的"稳定器"——过去3年ROE在23.7-24.1%之间，几乎纹丝不动。股息率常年超过5%，是典型的高分红公司。它是芒格式的"合理价格买好公司"的候选。

**风险：** 非洲猪瘟等疫情风险。行业增长率低（随人口增长）。负债率偏高（45%）。

### 第9名：海天味业（603288）

**核心数据（2024→2025）：**
- ROE: 20.2% → (2025: not in top 50 cutoff, likely lower)
- 负债率: 23.1% → ?
- 2024年营收: 269亿 | 净利润: 64亿
- 净利率: 23.9%

**护城河：品牌 + 渠道 + 口味锁定**

海天是中国酱油和调味品行业的绝对龙头。在中国家庭的厨房里，海天的产品渗透率超过90%。调味品生意的美妙之处在于：**一旦你用惯了一个品牌的酱油，你不会轻易换**。口味惯性是一种极强的护城河。

海天的渠道覆盖了全国几乎所有超市、便利店和农贸市场，深度下沉到县城和乡镇。这在调味品行业是一种"基础设施"级别的优势——新品牌可以铺货一线城市，但铺到全国100万个终端需要20年。

**定价权测试：** ✅ 海天每隔一两年就会小幅提价（3-5%），消费者几乎感觉不到。酱油在家庭支出中的占比极小，价格弹性几乎为零。

**风险：** 2025年数据不明显（未进前50），可能增速大幅放缓。食品安全事件是最大风险。消费者口味变化（低盐、健康趋势）可能影响传统酱油消费。

### 第10名：迈瑞医疗（300760）

**核心数据（2023→2024→2025）：**
- ROE: 34.7% → 28.8% → 19.7%
- 负债率: 30.4% → 28.0% → 27.4%
- 2025年营收: 333亿 | 净利润: 85亿
- 净利率: 33.1% → 31.9% → 25.4%

**护城河：中国医院基础设施级存在**

迈瑞是中国最大的医疗设备企业。其监护仪、超声、体外诊断等产品覆盖了中国90%以上的三甲医院。医疗设备行业的特点是：**进了医院就不会被换出来**。一台迈瑞的监护仪进入ICU后，后续的耗材、维护、升级都会继续采购迈瑞的产品。这是典型的"剃须刀+刀片"模式。

迈瑞的研发投入产出比极高——过去5年研发投入超过200亿，已经建立起一个覆盖全球的专利护城河。在海外市场，迈瑞是发展中国家医疗设备市场的领导者，正在向发达国家渗透。

**定价权测试：** ✅ 医疗设备的定价权强于普通消费品。医院不会因为便宜5%就换掉一个用了10年的设备品牌——培训、维护、数据对接的转换成本太高了。

**风险：** ROE从2022年的34%一路降到2025年的20%，趋势令人担心。原因可能是大规模增发/融资稀释了股东权益，或利润增速放缓。集采政策可能影响利润率。

## 巴菲特的最终建议

如果让我从这10家中**只选一家**——贵州茅台。它是我在A股能找到的最接近"完美的生意"。

但记住我的原则：**伟大的企业不等于是伟大的投资，除非你以合理的价格买入。** 即使是最好的生意，如果你出价太高，也可能多年不赚钱。耐心等待好价格，然后用桶而不是汤匙去买入。
